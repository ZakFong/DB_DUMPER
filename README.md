這是一套用 php-desktop 作為基底寫的資料庫匯出機
-------------------------------------------------
此程式是方便工程師提供 Table schema 給 SA，點一點就可以匯出
來，不用花很多時間維護 word、excel 檔案了 :D


程式說明：http://3wa.tw 資料庫匯出機
作者：羽山秋人                      
版本：V1.0
版權：GPL & MIT 喜歡就拿去用吧

目前支援資料庫類型：

（✔）MySQL、同MariaDB
（✔）MSSQL
（✔）PostgreSQL
（✔）Oracle
（ㄨ）SQLite (還沒寫完...~_~)
